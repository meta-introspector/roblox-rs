# roblox-rs-gui

A reactive GUI framework for Roblox, inspired by [Svelte](https://svelte.dev/).

## Vision

- **Declarative, reactive UI** for Roblox games, using idiomatic Rust.
- **Signals and reactivity**: State changes automatically update the UI.
- **Virtual DOM**: Efficiently diff and patch Roblox GUIs.
- **Component system**: Compose UIs from reusable components.
- **Type-safe**: Leverage Rust's type system for safe, robust GUIs.

## Planned Features
- Svelte-like component syntax (in Rust)
- Reactive signals and stores
- Virtual DOM diffing and patching for Roblox GUIs
- Event binding and two-way data binding
- Animation and transitions
- Integration with roblox-rs ECS and core crates

## Status
This crate is in early development. Contributions and ideas are welcome!
